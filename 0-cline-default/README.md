## 基本方針

どのような指示に対してAIがどのような結果を返したか,
さらに同じ結果を再現できるようにするため,
`prompts`ディレクトリに`日付-日時.md`のファイル名フォーマットにしたがって,
同じ結果を再現できるようなプロンプトと処理概要を都度保存する.
特に長いプログラムの詳細は処理概要に含めなくてよい.
図・表が必要な場合は`mermaid`など適切なフォーマットで記録する.
